How to play

Find the 4 digit number in less possible iterations, where face value and place value gives the hint for the number.

Face value count the number of correct digits and Place value count the correct position of digits.

Levels.

  • Easy: Boxes turns green if something matches to the provided digits and displays as you get its correct position.
  • Medium: Boxes turns green if something matches to the provided digits but number displays after perfect match of 4 digit number.
  • Difficult: Find digits without any hints.
